## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for API features
- Android device (for mobile app)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/ahxanbattle-crypto/Ahxanbattle.git
cd Ahxanbattle
```

2. **Open in browser**
```bash
# Simply open index.html in your browser
# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```

3. **Deploy to GitHub Pages**
```bash
# Push to main branch
git add .
git commit -m "Deploy AhxanBattle website"
git push origin main
```

Your website will be live at: `https://ahxanbattle-crypto.github.io/Ahxanbattle/`

## 📱 Mobile App

### Download APK
- File: `Ahxan battle.apk`
- Size: ~45-50 MB
- Min Android: 5.0 (API 21)
- Target Android: Android 14

### Installation Steps
1. Download the APK file
2. Enable "Unknown Sources" in settings
3. Tap the APK to install
4. Grant required permissions
5. Launch from app drawer
